REAPER中take和item有什么区别

发表于 2023-07-17

前言

刚开始使用reaper时可能会不太清楚take和item的区别, 这里给大家做简单的介绍

Take

Take (音频片段）是指在一个音频轨道上录制或导入的音频片段。当你在Reaper中录制或导入音频时，每个音频片段都被称为一个 Take。它代表了单独的音频录音或文件。

  • 一个轨道可以包含多个 Takes，在同一时间点上相互叠加或连续播放。
  • 每个 Take 可以具有不同的长度、音量、效果处理等属性。
  • 在混音或编辑过程中，你可以切换和比较不同的 Takes，选择最好的版本或进行组合。

Item

Item 项目）是指 Reaper中的一个音频片段的可视表示，它将 Take（音频数据）与其它参数（如位置、长度、音量、效果等）结合起来。

  • 一个音频片段（Take）可以有一个或多个关联的Items，每个Item都显示了该音频片段的可视化内容和相关的控制参数。
  • 通过编辑 Item，你可以移动、调整大小、裁剪、应用效果、平衡音量等对音频进行精确控制。
  • 在项目中，你可以使用项（Item）进行整体的编辑、处理和排列。

总结

总之，Take是音频片段本身，而Item 是展示了 Take 的可视化表示，并允许你在项目中对其进行操作和调整。

通俗点说, Take就相当于是音频素材库,Item就相当于是一个微型音频工程, 它只是对Take进行了引用. 这也是为什么我们在Reaper中对某个item无论怎么剪辑和复制, 工程目录下都不会新增音频文件

Take和item的这种设计不仅大大减少了工程体积, 还使得音频处理变得更加的自由灵活

在REAPER中如何建立音频参考轨道

发表于 2023-07-13

需求

reaper中如何建立参考音频轨道 其信号绕过master输出 实现当前混音和参考音频切换播放呢?

我们在混音过程中可能会拿某个同类型的商业混音作品来进行参考, 使我们的混音更加的理性

在其他DAW中实现起来有些费劲, 但是在REAPER中却非常容易, 具体实现步骤如下

实现步骤

  1. 打开Reaper，并确保你的项目已加载。

  2. 在Reaper的轨道窗口右键单击空白区域，选择Insert Track（插入轨道）或使用快捷键Ctrl+T（Windows）/Cmd+T（Mac）

  3. 打开调音台, 在新建的轨道上，右键单击并选择弹出路由设置对话框

  4. 在弹出的对话框中，找到Master send选项，并取消勾选

  5. 同时, 找到Hardware Output（硬件输出）选项，并选择你Output 1/Output 2输出。这是不同于主输出的输出通道。

  6. 现在，你的参考音频轨道已创建，通过M/S则可来回切换对比试听了

混音和参考音频快速切换播放

混音过程中, 需要频繁切换参考轨道的播放, 而每次切换都需要找到相关轨道来回对其M/S, 效率非常的低而且繁琐, 有没有高效的方法呢, 比如一键切换?, 这里给大家分享两个小技巧:

第一种 使用grouping编组

  1. 首先选中master轨和reference, 右键选择Track Grouping Parameters..进入编组设置窗口
  2. 将Mute Lead和Mute Follow同时勾选上

  1. 此时我们Mute其中任意一轨, 另一轨也跟着被Mute了, 此时我们按住shift键左键取消其中一轨的Mute, 这时我们再点击Mute就能实现两轨之间反向Mute了 , 如下图:

第二种 使用Action组合

虽然第一种方式能有效减少切换的流程, 但我还是感觉有些慢, 于是决定使用自定义Action的方式, 这样可以给其添加快捷键

  1. 首先打开Action List窗口, 创建一个新的Action

  2. 然后搜索Toggle mute for master track和Toggle solo for track 01并将它们拖拽至右方, 这里我将自定义Action命名为toggle reference

  3. 为了方便使用 我为其添加了快捷键/

  4. 此时回到轨道面板, 我们只需要将参考音频放置在第一轨道上, 并且Mute它, 后面我们就可以随时通过快捷键切换音频的播放了

如何使REAPER中的仪表和波形看起来更加平滑

发表于 2023-07-12

在本期快速教程中，我将向您展示如何使 REAPER 中的仪表和波形在播放时看起来更加平滑👌

REAPER 允许您自定义仪表和波形的平滑度。默认情况下，设置非常低，导致出现锯齿状外观。

我现在将介绍如何自定义其中每一个。

🌊 波形更平滑

您可能已经注意到，当 REAPER 录制时，波形绘制得非常慢。这是因为它的默认刷新率仅为 3 Hz。我们可以将其设置为 30 Hz，并使其看起来更加平滑，如下所示：

设置步骤：

  1. 首先进入到Preferences > Recording。

  2. 然后找到Show preview ...update frequency一栏改为30

  3. 完成！尝试录制，你的波形应该看起来更好😊

🎚 更平滑的仪表

VU Meter也可以可调得更加顺滑！

设置步骤

  1. 首先进入到Preferences > Track Control Panels

  2. 然后找到Meter update frequency一栏, 将其设置为30 Hz, 同时将Meter Decay 设置为40 db/sec

  3. 完毕！您的轨道计现在看起来应该更平滑了。

如果您的轨道上没有VU表, 可以参考之前的文章:《在REAPER中如何实现在轨道上显示VU表》

就是这样！

如果您发现任何性能问题，您可以将频率数字降低到15或更低。即使在较旧的机器上，我个人也没有注意到这些设置有任何差异。

享受💙

在REAPER中如何实现在轨道上显示VU表

发表于 2023-07-12

介绍

这是ZenoMOD制作的一个很棒的JS 插件，它允许您将可定制的 VU 仪表嵌入到您的轨道中。

您可以在轨道面板或混音器面板中显示它，并且有超过 10 种皮肤变化！

我最喜欢的功能是可以直接在调音台中更改电平。

单击并拖动MCP / TCP来调节音量。并双击将音量重置为默认值。

这很方便我们给音轨做Gain Staging

如何安装

  • 如果你安装了Reapack，你的 FX 列表中应该有这个插件, 插件名叫VU Meter如果你没有Reapack，可以参考我之前发的文章《REAPER宿主功能增强秘技之脚本和插件》
  • 或者，如果您需要此插件的原包，也可以在[REAPER stash](https://stash.reaper.fm/v/42223/VU Meter (ZenoMOD))中直接下载。

如何使用

  1. 在Reapack仓库中搜索VU Meter并安装
  2. 安装后，将VU Meter (ZenoMOD)加载到您的任何轨道。
  3. 在调音台面板中，右键单击插件，然后勾选Show embedded UI in MCP（当然你也可以在 TCP中显示它）
  4. 如何要将其设置为所有新轨道的默认值，实现新插入的轨道自动添加该VU表插件, 请打开FX Track window > Right click the plugin > FX Chains > Save all FX as default chain for new tracks…

Zeno 的 VU Meter 是今年REAPER 推出的最棒的产品之一，它使gain staging的方式更加简单，同时看起来也非常好！✨

混音中提升音轨融合度之均衡处理

发表于 2023-07-12

前言

混音过程中影响融合度的因素有很多, 之前给大家介绍过共同元素理论, 具体可查阅之前文章:《混音中各音轨共同的元素越多融合度越高》

今天 我们主要来研究讨论一下均衡器对声音融合度的影响

从个人经验上, 主要分为以下几个点:

  • 衰减特色频率
  • 提升某个频段达到对其他乐器的贴合
  • 多个音轨间使用同类型的的均衡效果器

特色频率衰减

衰减声音的特色频段能让声音听起来更温和, 虽然少了一分刺激感, 但是和其它音轨更加的融合了

这个世界 大部分都是不起眼的普通人, 特立独行且耀眼的终归是少数, 太过耀眼自然很难融入群众

混音也是同样的道理, 衰减声音中让你觉得兴奋刺激的频段, 这样能使整体更加地平衡

除此之外, 声音中如果存在一些让人不舒服的共振频率, 也需要去除, 这能让声音听起来更加清晰有质感, 具体方法是通过扫频或者solo某个频率点, 找出腔体感特别明显以及太过刺耳的频率点并衰减它

衰减频率这一步骤建议使用窄q值的数字均衡器, 比如waves Q系列插件或者肥波插件, 它们能精准地控制住频率

提升频段

提升频段, 建议使用宽Q值的建模均衡器, 主要用来调整声音的色彩, 让其变得更暗还是更亮, 从而实现和其它音轨的贴合

同时 建模EQ的染色能给声音提升质感, 增加一点点可望不可即的感觉

具体操作方法是, 通过宽Q值的扫频方式, 一边扫, 一边仔细听需要进行融合的音轨, 如果是扫人声那就听伴奏. 当我们扫到某个频段, 伴奏突然变得更清晰了, 那么这个点差不多就是融合点, 提升它能让声音更加融合

有时候, 你可能会发现不止一个融合点, 此时我们就需要听当前扫频的声音音色, 选择一个你觉得好听的就行, 这样 既保证了融合度, 又不至于声音变得难听

原理分析

我个人利用频谱分析仪分析两个音轨后得出粗浅结论:

当两个音轨频谱趋于一致的时候, 融合度能得到大幅提升

比如有两个音轨, 一个高低频很饱满, 另一个低频足够 但是高频不足, 那么这两个音轨合在一块的听感就是一个很亮, 一个很暗, 虽然声音不打架了, 但两者分离感很明显, 它们就犹如不同世界的两个人,很难融合

此时我们将高频不足的那个声音做一个高频提升后, 就能明显感觉到融合度的提升

我认为这个原理也可以归纳到共同元素理论当中

使用同类型的的均衡效果器

这一点其实就是依照共同元素理论了, 同类型的效果器染色一致, 能有效增加融合度

效果链

依照上面的所说, 我在对声音进行均衡处理时, 通常依照以下顺序:

  • 减法EQ
  • 加法EQ
  • 压缩器
  • ….

总结

本文中, 关于关于均衡器对提升融合度, 主要分为以下两大方面:

  1. 增加共同元素: 同类型效果器, 频谱趋于一致….等等
  2. 衰减特色频率: 平凡但不平庸

注意事项

  1. 我们在扫频时不建议将音量提的过高, 否则你会发现到处都是不舒服的频率, 一般不超过12db
  2. 衰减的点不要太多, 过多反而会适得其反, 一般来说 这种情况的发生可能是操作方法不对
