在REAPER中如何快速将 MIDI 柱式和弦修改成分解和弦模式

这是一个简单的鼠标修改器，您可以在 REAPER 中设置，为您的 MIDI 和弦提供弹奏的感觉🎹

如果您正在处理钢琴、吉他或任何其他可以弹奏的乐器，这非常有用。增加了很多真实感，并且可以改变很多氛围。

设置

  1. 打开Preferences → Mouse Modifiers → context: MIDI note – left drag
  2. 双击您要使用的修饰符（我将其设置为Ctrl ）。
  3. 选择 Stretch notes (arpeggiate/chordize) > Just stretch

就是这样！

现在，每次按住所选的修改器时，鼠标光标都会发生变化，让您知道您处于和弦化模式。

如何实现在同一台机器上安装多个REAPER

发表于 2023-07-11

REAPER 允许您使用便携式安装来安装任意数量的 REAPER！🤯

这意味着您可以拥有多个具有不同配置的 REAPERS 以用于不同的目的。它可以是一个用于配音的，另一个用于编曲的，一个用于视频编辑的，还有一个主用的。

我喜欢让每一个都有自己的快捷键、主题、屏幕设置以及任何对相应工作流程有意义的偏好设置。

不仅如此，它还可以用于在闪存驱动器上安装 REAPER，并随时随地携带您的整个设置！

什么是 REAPER 便携式安装？

使用“便携式安装”选项安装 REAPER 会创建一个包含程序、配置、主题、热键设置等的独立目录。

这有几个优点💪

  • 您可以将 REAPER 放在闪存驱动器上并随身携带以在其他计算机上使用。这意味着您可以随身携带整个 REAPER 设置。
  • 您可以拥有不同的 REAPER，并针对不同的工作流程进行特定设置。
  • 您可以拥有不同版本或设置的 REAPER，而不会相互妨碍或妨碍您通常的Reaper 安装。使用完其中一个后，只需删除该文件夹即可，不会对其他安装产生任何影响。
  • 通过复制便携式安装文件夹轻松备份 REAPER 或创建实验副本。
  • 使用默认设置重现便携式安装中的错误，以帮助开发人员调试。

如何便携式安装REAPER

📓在 Windows 上

  1. 下载REAPER并打开安装程序。

  2. 启用Protable install并单击浏览

  3. 选择或创建一个要安装 REAPER 的新文件夹。

  4. 安装后，打开文件夹并运行REAPER.exe 。这是 REAPER 的便携式版本。

🍎 在 Mac 上

  1. 在 Mac 上要安装REAPER的任意位置创建一个新文件夹。按照您想要的方式命名。

  2. 下载REAPER并打开.dmg文件。然后将REAPER.app复制到您在步骤 1 中创建的文件夹中。

  3. 在 Mac上打开TextEdit应用程序并创建新文档。并将其保存重命名为reaper.ini。确保禁用底部的if no extensionn is provided, use .txt

  1. 将您刚刚创建的reaper.ini”文本文件放入您在步骤 1 中创建的便携式文件夹中。您的文件夹应如下所示：

  2. 最后，运行该REAPER 。这将自动创建所有配置文件。

建议

将REAPER.exe/REAPER.app文件保留在单独的文件夹中非常重要。这样，它将保持可移植性并使用该文件夹内的任何配置。

您可以创建REAPER.exe/REAPER.app的桌面快捷方式，以便更轻松地访问。如果您想将其用于特定目的，我还建议您重命名它。

安装后

  • 您需要将许可证密钥添加到便携式安装中。要查找文件reaper-license.rk ，请打开正常的 Reaper安装并选择Options → Show REAPER resource path in explorer/finder...。
  • 如果你想将配置从一个Reaper 转移到另一个Reaper中, 那么可以：Preferences → General → Export/Import configuration
  • 如果您使用 SWS 扩展或Reapack，则还需要单独安装它。这是唯一没有从上述步骤中导出的内容。点击Options → Show REAPER resource path in explorer/finder...在资源目录中找到UserPlugins文件夹进行拷贝即可。

就是这样！

现在您已经知道如何通过便携式安装来安装 REAPER，您可以拥有用于不同目的和工作流程的多个版本的 REAPER。您还可以将 REAPER 安装在闪存驱动器中并随身携带以在其他计算机上使用。

在REAPER中显示音频的5种方法

发表于 2023-07-10

前言

REAPER 的一项独特功能是能够以五种不同的方式显示音频项目！🤯

这确保了无论需要处理什么样的音频，REAPER都能为用户提供完美的视觉呈现👌

我现在将回顾它们中的每一个，演示它们的用途，并提供显示它们的快捷方式。

1. 规律的高峰

这些是我们大多数人都熟悉的默认波形。

我们可以清楚地直观地了解音频的响度和长度。

🎯要为此视图分配快捷方式或工具栏，可以在Action List中搜索Peaks: Show normal (non-spectral/non-spectrogram) peaks

2. 纠正峰值

就像常规峰值一样，但您只能看到波形的一半。

这通常用于视频编辑，因为它有助于更好地识别对话和更安静的段落。如果您像大多数视频编辑器工作区一样垂直空间有限，也很有用~

🎯要为此视图分配快捷方式或工具栏，可以在Action List中搜索Peaks: Rectify peaks

3. 光谱峰

此视图允许您将不同的频率可视化为颜色。

这非常适合检测人声中的嘶嘶声、录音中的噪音、转录音符等！

🎯要为此视图分配快捷方式或工具栏，可以在Action List中搜索Peaks: Toggle spectral peaks

⚙️您还可以搜索并使用以下操作调整设置：View: Show peaks display settings

4. 频谱图

这是频率强度的视觉表示。

我们可以比较 200 Hz 和 10 kHz 等不同频率下的能量水平，并了解这些水平如何随时间变化。

🎯要为此视图分配快捷方式或工具栏，可以在Action List中搜索Peaks: Toggle spectrogram

⚙️您还可以搜索并使用以下操作调整设置：View: Show peaks display settings

5. 频谱图和峰值

这将显示一半频谱图和一半峰值。

我经常使用它，因为它提供了两全其美的优点！您可以查看常规响度峰值以及其上方的频率信息，从而更容易识别声音。

🎯要为此视图分配快捷方式或工具栏，可以在Action List中搜索Peaks: Toggle spectrogram and peaks

福利：光谱编辑

使用任何频谱图视图时，您可以搜索并使用Spectrogram: Add spectral edit to item操作来调整音频Izotope RX 风格！非常好。我通常用它来消除噪音和异响。

就是这样！

如果您为它们设置快捷方式，您可以在它们之间快速切换，并以最适合每个任务的方式查看音频。或者您可以将它们放置在顶部工具栏之一。

REAPER中实现轨道自动着色

发表于 2023-07-09

介绍

此功能使您能够自动更改轨道/片段/标记的颜色、添加图标以及根据您可以设置的规则修改MCP 和 TCP 布局！🚀

例如，您可以设置一条规则，每当您制作名为Guitar的曲目时，它都会自动将其着色为红色并添加吉他图标，如下所示：

超级棒对吧！？🤩

我也喜欢自动着色区域！我喜欢给我常用的歌曲结构分配不同的颜色，比如“主歌”、“副歌”、“桥段”、等等……

让我引导您了解如何安装以及您可以用它做的所有事情。

如何安装

  1. 下载并安装SWS 扩展。
  2. 打开REAPER 并在扩展下拉菜单中选择自动颜色/图标/布局。
  3. 单击“选项” ，然后启用您需要的所有选项。（自动轨道着色、自动轨道图标… ）

自动颜色/图标轨道

  1. 单击添加规则。
  2. 双击“（名称）”的位置，然后输入要设置颜色的曲目的名称。
  3. 右键单击旁边显示“无”的位置，单击“设置颜色”，然后为轨道选择一种颜色。
  4. 通过右键单击颜色列中的空白区域来放置图标。

让子级追踪与父级相同的颜色

  1. 单击添加规则。
  2. 右键单击显示“（名称）”的位置。
  3. 选择“（子级）”。
  4. 右键单击颜色列中显示“无”的位置，然后选择“父级”。

自动为区域着色

  1. 单击添加规则。
  2. 右键单击显示“轨道”的位置。
  3. 选择“地区”。
  4. 双击显示“（名称）”的位置，然后输入要设置颜色的区域的名称。
  5. 右键单击其旁边显示“无”的位置，单击“设置颜色”，然后为该区域选择一种颜色。

记住…

自动着色列表按优先级排序，您可以在第一列中看到数字。数字越小优先级最高。

例如，如果您将“吉他”设置为黄色，作为列表中的#1，将“原声吉他”设置为红色，作为列表中的#2，则它将根据优先级较高的颜色为曲目着色。在这种情况下，只需将“原声吉他”拖到列表中，使其优先级高于“吉他”曲目即可。这将导致名为Acoustic Guitar的曲目显示为红色，而名为Guitar的曲目将显示为黄色。

REAPER中十多个包络自动化技巧

发表于 2023-07-08

前言

如果您希望在 REAPER 中提高自动化效率，我整理了一些有用的便捷技巧。🚀

1.快速建立包络

按键盘V打开/关闭音量包络通道, 按住shift配合鼠标左键即可快速画出大致包络曲线。

按键盘P打开/关闭声像包络通道。

2. 四点包络线

创建时间选择并按Shift + Ctrl/Cmd创建四点包络。

这非常适合快速调整某个部分的值，同时保持通道的其余部分不变。

💡

提示：默认情况下，它将产生非常短的过渡曲线，这听起来可能有点刺耳。您可以通过进入“首选项” > “包络显示”并设置“自动创建包络边缘点的过渡时间”来 调整曲线的平滑度

3. 调整包络曲线

按住Alt/Opt 的同时单击并拖动包络曲线以自由塑造其形状。对于平滑过渡非常有用。

按住Alt/Opt的同时双击鼠标左键可将其重置。

4.手动调整点

按住Ctrl/Cmd并双击包络点可手动输入值或更改曲线形状。

5. 创建自动化项目

按Alt/Opt并在包络通道底部拖动以创建自动化项目。

🤔什么是自动化项目？

它们是包络的容器。您可以像常规项目一样循环、复制、拉伸和池化。您基本上可以将您的自动化视为物品！

（您也可以双击它们来应用 LFO 并调整设置！）

6. 微调包络点

按住Ctrl/Cmd + Alt/Opt并上下拖动点，以非常小的步长进行调整。

7.快速删除包络点

按Alt/Opt 键并单击包络点将其删除。

8. 重置包络点

双击包络点可将其值重置为 0dB。

9. 画包络

在包络通道上拖动时按住Ctrl/Cmd以徒手绘制，忽略捕捉。

💡提示：您可以对其进行设置，使其贴合您的网格。
为此，请转至首选项>鼠标修改器>上下文：包络通道 | 左键拖动并将鼠标修改器设置为“徒手绘制信封”

💎 其他快捷技巧

接下来的操作没有快捷方式，但它们非常有用，我建议为它们设置一个！

10.显示和隐藏包络通道

如果您有一堆包络占用了会话中的空间，可以通过以下方法快速显示/隐藏所有信封或仅显示选定的曲目。

在菜单的Actions List中，搜索Envelope: Toggle show all envelopes for all tracks。我将其快键键设置为Alt + Tab。

11.显示最后编辑的包络通道

这是我最喜欢的动作之一！

您无需查看大量参数即可启动包络通道。只需触摸您需要的参数，运行此操作，该参数的通道就会出现在轨道列表上！

在菜单的Actions List中，搜索FX: Show/hide track envelope for last touched FX parameter, 并为其设置快捷键。我把它设置为L 。

就是这样！
